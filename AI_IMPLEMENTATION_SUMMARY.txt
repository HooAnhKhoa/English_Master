📝 API ENDPOINTS

POST /api/v1/ai/conversations
  → Start new conversation
  → Returns: conversationId, firstMessage, suggestedResponses

POST /api/v1/ai/conversations/:id/messages
  → Send text or audio message
  → Returns: reply, correction, vocabulary, suggestions, score, xp

POST /api/v1/ai/conversations/:id/end
  → End conversation & get scorecard
  → Returns: overallScore, totalTurns, duration, commonErrors, xp, badges

GET /api/v1/ai/conversations/:id
  → Get conversation history

GET /api/v1/ai/conversations
  → List all user conversations (paginated)

POST /api/v1/ai/analyze
  → Analyze text grammar (standalone)
